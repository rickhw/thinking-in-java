### 🎮 遊戲開發概述
- 本影片由 RyiSnow 主持，重點介紹如何製作一款 **2D 動作角色扮演遊戲** 的 **世界地圖**。

### 🗺️ 世界地圖設計
- 遊戲地圖為 **50x50** 的規模，具備更大的行走空間，並可根據玩家需求自由調整。
- 添加了三種額外的磚塊：**木磚**、**沙磚** 和 **土磚**，並已成功載入（使用於遊戲中）。

### 👨‍💻 程式碼調整
- 修改 **Player 類別**，使玩家始終固定在螢幕中央，並改變圖塊繪製的循環範圍，以顯示整個世界地圖。

### 📐 座標系的劃分
- 進行了 x 和 y 的重新命名，使其清晰可辨：**worldX** 和 **worldY** (地圖座標)，**screenX** 和 **screenY** (螢幕座標)，以便有效管理相機功能。

### ⚡ 繪圖效率
- 為提升性能，設置了 **條件分支**，只繪製可見範圍內的圖塊，減輕了繪圖負擔，避免在螢幕上不必要的顯示。

### 🔄 下一步計劃
- 雖然目前已經成功能夠探索世界地圖，但仍缺乏**碰撞檢測**功能，下一集將處理這一問題。
